# BiliTogether

一个基于 `Chrome Manifest V3` 的双人一起看 B 站插件原型，特点：

- 纯前端，无 Supabase、无房间服务
- 手动复制粘贴连接码 / 确认码建立连接
- 基于 `WebRTC DataChannel` 同步播放控制和聊天
- 支持播放、暂停、拖动、倍速、切换视频
- 双方都可以抢控制，按最后一次操作生效

## 本地加载

1. 打开 `chrome://extensions/`
2. 启用“开发者模式”
3. 选择“加载已解压的扩展程序”
4. 选择当前仓库目录 `e:\code\BiliTogether`

## 使用方式

1. 房主点击“生成连接码”，会自动复制
2. 加入方贴入连接码，点击“生成确认码”，也会自动复制
3. 加入方把确认码发回房主
4. 房主贴入确认码，点击“完成连接”
5. 双方打开 B 站视频页后即可同步播放和聊天

## 当前限制

- 只支持 `Chromium` 浏览器
- 只支持 `2 人`
- 不支持自动重连，断开后需要重新配对
- 使用公共 `STUN`，某些网络环境可能无法成功建立 P2P 连接
- 这是首版原型，未包含自动化测试和发布配置
